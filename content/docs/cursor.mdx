### 2. 提示词指南

<Callout type="info" title="什么是提示词？">
  提示词（Prompt）是你与 Cursor 沟通的方式。它决定了 Cursor
  代码输出的质量。一个好的提示词可以让代码生成更加准确，相反则会得到你不想要的代码。
</Callout>

<Cards>
  <Card title="❌ 不好的提示词" type="error">
    "帮我写一个登录页面。" 
    这样说，Cursor 可能生成一个默认的 HTML文件，功能简单，样式也很普通。
  </Card>

  <Card title="✅ 好的提示词" type="success">
    "请为我生成一个 Vue3 登录页面，带有表单验证、响应式布局，并使用
    TailwindCSS。" 
    这样 Cursor 才可以为你生成你真正想要的代码。
  </Card>
</Cards>
