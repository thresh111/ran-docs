## 快速查询

有一个字典，返回最大下标

```js
function Words(wordList) {}

const words = new Words(['apple', 'banana', 'orange', 'pear', 'nana']);

words.search('a', 'e'); // 0
words.search('b', 'a'); // 1
words.search('or', 'ge'); // 2

words.search('pear', ''); // 3
words.search('xx', 'xx'); // null

words.search('', 'nana'); // 4
```
